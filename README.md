‘# Sprint_5’
Stellar Burgers - UI Autotests

Описание проекта
Автоматизированные тесты для веб-приложения Stellar Burgers, покрывающие функционал:
- Регистрация
- Авторизация
- Навигация
- Личный кабинет

Что тестируем
# 1. Тесты регистрации
- ✔ Успешная регистрация с валидными данными
- ✔ Ошибка при регистрации существующего пользователя
- ✔ Ошибка при некорректном пароле
- ✔ Ошибка при пустом имени

# 2. Тесты авторизации
- ✔ Вход через кнопку "Войти в аккаунт" на главной
- ✔ Вход через "Личный кабинет"
- ✔ Вход через ссылку в форме регистрации
- ✔ Вход через ссылку в форме восстановления пароля
# 3. Тесты навигации
- ✔ Переход в личный кабинет
- ✔ Возврат в конструктор из ЛК
- ✔ Переход по логотипу
- ✔ Переключение разделов конструктора (Булки, Соусы, Начинки)

# 4. Тесты личного кабинета
- ✔ Выход из аккаунта
- ✔ Восстановление пароля (базовая часть)

# Технологии
- **Python 3.9+**
- **Pytest** - фреймворк для тестов
- **Selenium WebDriver** - автоматизация браузера
- **WebDriver Manager** - управление драйверами
